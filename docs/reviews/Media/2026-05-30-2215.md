- **File**: src/Media/Media.Api/Application/CompleteMultipartUpload.cs:85
- **Issue**: Uses `await context.Database.BeginTransactionAsync(ct)` inside MassTransit handler
- **Impact**: Violates the MassTransit Entity Framework Core Inbox/Outbox pattern, can cause nested transactions and bypass outbox guarantees

// MassTransit already wraps the consumer in a transaction
