## CRITICAL: Service Token Endpoint Missing Rate Limiting
- **File**: src/Identity/Identity.Api/Controllers/AuthenticationController.cs:208-229
- **Issue**: Service token endpoint lacks rate limiting, vulnerable to brute force attacks on the shared secret
- **Impact**: Attackers could brute force the service secret to gain service-level access
- **Fix**: Add rate limiting attribute:
```csharp
[HttpPost("service-token")]
[AllowAnonymous]
[EnableRateLimiting("service-auth")] // Add this
[ProducesResponseType(StatusCodes.Status200OK)]
public async Task<IActionResult> ServiceToken(/*...*/)
```
- **Confidence**: 9/10

## MEDIUM: Global Query Filter Could Hide Required Data
