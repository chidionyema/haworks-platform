## CRITICAL: Hangfire Dashboard Authorization Logic Error
- **File**: src/Scheduler/Scheduler.Api/Program.cs:95-105
- **Issue**: The `HangfireLocalRequestFilter` requires BOTH local connection AND admin role. Local requests in development won't have authentication.
- **Impact**: Dashboard inaccessible in development, potential security bypass in production if local network is compromised
- **Fix**: 
```csharp
public bool Authorize(DashboardContext context)
{
    var httpContext = context.GetHttpContext();
    var connection = httpContext.Connection;
    bool isLocal = connection.RemoteIpAddress != null
        && (connection.RemoteIpAddress.Equals(connection.LocalIpAddress)
            || System.Net.IPAddress.IsLoopback(connection.RemoteIpAddress));
    
    // In development: allow local access OR authenticated admin
    // In production: require authenticated admin only
    if (context.GetHttpContext().RequestServices
            .GetService<IHostEnvironment>()?.IsDevelopment() == true)
    {
        return isLocal || httpContext.User.IsInRole("admin");
    }
    return httpContext.User.IsInRole("admin");
}
```

## CRITICAL: Division by Zero in VaultLease.NeedsRotation()
- **File**: src/Scheduler/Scheduler.Domain/Entities/VaultLease.cs:88-98
- **Issue**: If `ExpiresAt <= CreatedAt`, division by zero occurs: `var ratio = elapsed / totalTtl;`
- **Impact**: Runtime exception causing service crashes during rotation checks
- **Fix**: Add guard clause:
```csharp
public bool NeedsRotation(double thresholdPercent = 0.8)
{
    if (Status != VaultLeaseStatus.Active)
        return false;

    var totalTtl = ExpiresAt - CreatedAt;
    if (totalTtl <= TimeSpan.Zero)
        return true; // Immediately needs rotation if TTL is invalid

    var elapsed = DateTimeOffset.UtcNow - CreatedAt;
    var ratio = elapsed / totalTtl;
    return ratio >= thresholdPercent;
}
```
